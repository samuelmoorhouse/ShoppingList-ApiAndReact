# Simple Cross Platform Shopping List Using ASP.NET10 Api And React
This application is aimed to create a simple shopping list app while I learn some new or updated technologies, such as using .net 10 api instead of my previously learned mvc and then creating a separate frontend in react and making it cross platform. 
This is very different from how I have learnt to do asp.net using the model view controller earlier this year. The new or updated tech I want to use and learn more of in this project is C# with ASP.NET 10 for the backend with TypeScript and React for the frontend. 
In this project I will also document and design in the same way I have in university, but I will be learning Figma to create my clickable wireframe instead of mockplus as it is the industry standard.
